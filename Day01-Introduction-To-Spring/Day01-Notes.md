# Day 1 - Introduction to Spring Framework

## What is Spring?

Spring is a Java Framework used to build Java applications easily.

## Why Spring?

Spring reduces code complexity and helps create loosely coupled applications.

## Tight Coupling

When one class directly depends on another class.

## Loose Coupling

When classes are independent and dependencies are provided externally.

## IoC

Inversion of Control means Spring creates and manages objects instead of the programmer.

## Bean

An object managed by Spring is called a Bean.

## IoC Container

The container responsible for creating and managing beans.
